Work has been Hosted on https://bryckel-frontend.vercel.app/

For running locally, the following environment variables have to be set for the backend:
STATUS="DEV"
DEV_SECRET_KEY=
DEV_DB_NAME=
DEV_DB_USER=
DEV_DB_PASSWORD=
DEV_DB_HOST=
DEV_DB_PORT=

And for the frontend:
REACT_APP_BACKEND_BASE_URL="https://hostname"
REACT_APP_GOOGLE_CLIENT_ID=
REACT_APP_GOOGLE_CLIENT_SECRET=
